# Patchstop
Disables, prevents & cancels any automatic updates from the Microsoft Store.

# Usage
- Download the latest release.
- Now if any update is triggered via the Microsoft Store, it will cancelled by Patchstop.
- To close Patchstop: 
  - Go to your notification area.
  - Click on the `⮝` icon.
  - Find Patchstop's `❌` icon.
  - Right click it and click on `[Exit]`.
